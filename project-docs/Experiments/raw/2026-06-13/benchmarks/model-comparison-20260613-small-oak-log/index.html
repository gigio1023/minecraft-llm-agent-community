Setup And Exclusions

  • All scored runs used the same fresh fixture and the world-scenario setup status was passed. A non-required compatibility gamerule, minecraft:spawn_chunk_radius, failed consistently and did not block setup.
  • During preparation, fixture setup had two real blockers: setworldspawn x y z yaw was invalid on this server version, and setup fills needed a forceload add. Both were fixed before scored runs.
  • ModelScope Qwen Max/Plus now pass follow-up tool-call transport checks with tools + tool_choice: auto; named tool calls also work with top-level chat_template_kwargs.enable_thinking=false. Qwen was not scored only because the repo runtime still lacks a wired modelscope-api social-cycle adapter.
  • gemma-4-31b-it JSON smoke succeeded, but the social-cycle run was stopped after several minutes waiting on the first goal_mind provider call. It is not included in the scored 3-model table.
