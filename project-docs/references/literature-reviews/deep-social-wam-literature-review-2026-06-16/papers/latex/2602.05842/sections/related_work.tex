\paragraph{Training Decision-Making Agents}
% general RL, SFT
LLM-based agents \citep{yao2023reactsynergizingreasoningacting,shinn2023reflexionlanguageagentsverbal} has seen wide applications in domains such as interactive gaming \citep{wang2023voyageropenendedembodiedagent,feng2025groupingrouppolicyoptimizationllm}; software engineering \citep{jimenez2024swebenchlanguagemodelsresolve,yang2024sweagentagentcomputerinterfacesenable}; computer, phone, browser-use \citep{xie2024osworldbenchmarkingmultimodalagents,rawles2025androidworlddynamicbenchmarkingenvironment,zhou2024webarenarealisticwebenvironment,yu2025exactteachingaiagents}, and more.
Many early work on training language agents primarily rely on imitation learning (i.e., SFT), using either demonstrations from human experts \citep{deng2023mind2webgeneralistagentweb,chen2025guicoursegeneralvisionlanguage,wang2025opencuaopenfoundationscomputeruse} or trajectories synthesized from stronger LLMs often accompanied with a set of manually designed workflows/heuristics \citep{zeng2023agenttuningenablinggeneralizedagent,chen2024agentflandesigningdatamethods,su2025learnbyinteractdatacentricframeworkselfadaptive,xu2025aguvisunifiedpurevision}.
% While high-quality SFT data provides dense supervision, it is challenging to scale such data due to the cost of collecting high-quality demonstrations.
While high-quality SFT data offers dense supervision signals, it is difficult to scale due to the high cost of collecting such demonstrations.
Alternatively, recent efforts in RL bypasses the need for step-by-step demonstrations and instead directly learn from terminal rewards (\ie{} task success) through trail and error.
Recent work include \citet{feng2025retoolreinforcementlearningstrategic,rllm2025,luo2025gui,jin2025search,ragen}, often powered by algorithms such as PPO \citep{schulman2017proximalpolicyoptimizationalgorithms} and GRPO \citep{shao2024deepseekmathpushinglimitsmathematical}.
% However, designing these terminal reward functions in complex environments still requires careful heavy involvement from human experts \citep{chowdhury2024swebenchverified,xie2024osworldbenchmarkingmultimodalagents,gou2025mind2web2evaluatingagentic}, and hence still faces scalability issues.
% Together, these work motivate the need for a more scalable training method to bridge the gap between models pretrained with next-token-prediction on internet-scale data to downstream applications in complex agentic environments.
However, designing task-success reward functions in complex environments still requires substantial human expertise \citep{chowdhury2024swebenchverified,xie2024osworldbenchmarkingmultimodalagents,gou2025mind2web2evaluatingagentic}, limiting scalability.
Together, these works motivate the need for more scalable training methods to bridge the gap between next-token-prediction pretrained models and their downstream applications in long-horizon agentic environments.

\paragraph{Training World Models}
Beyond task-success rewards, real-world interaction data contains rich information that can be used to help decision-making.
Early examples include Dyna algorithms \citep{dyna} which separately trains a world model to combine model-based with model-free learning for efficient policy training.

% However, these methods still either requires expensive computation to train two separate models (world model and policy model) or requires additional annotations from experts/stronger LLMs to generate high-quality data for world modeling.

Recent applications on LLM agents either train a \emph{separate} world model to support inference-time algorithms such as MCTS \citep{hao2023reasoninglanguagemodelplanning,wu2025rlvrworldtrainingworldmodels,chae2025webagentsworldmodels,gu2025llmsecretlyworldmodel}, or jointly learn world models and policies within a single model to improve generalization \citep{faircodegenteam2025cwmopenweightsllmresearch,zhang2025agentlearningearlyexperience,yu2025dynamindlearningsimulateexperience,yu2025dynathinksynergizingreasoningacting,feng2025webworldmodels,li2025wordworldlargelanguage,qian2026current}.
However, these approaches either require expensive training/inference of multiple models, or rely on additional annotations from experts/strong LLMs during world model learning.
We propose \wmrlshort{} as a scalable, \emph{self-supervised} method to improve the world knowledge and decision-making ability of a single model.
