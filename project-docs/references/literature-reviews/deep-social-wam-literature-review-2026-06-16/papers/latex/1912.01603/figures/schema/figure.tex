\caption[]{Components of Dreamer. (a) From the dataset of past experience, the agent learns to encode observations and actions into compact latent states (\iconstate), for example via reconstruction, and predicts environment rewards (\iconreward). (b) In the compact latent space, Dreamer predicts state values (\iconvalue) and actions (\iconaction) that maximize future value predictions by propagating gradients back through imagined trajectories. (c) The agent encodes the history of the episode to compute the current model state and predict the next action to execute in the environment. See \cref{alg:agent} for pseudo code of the agent.}
