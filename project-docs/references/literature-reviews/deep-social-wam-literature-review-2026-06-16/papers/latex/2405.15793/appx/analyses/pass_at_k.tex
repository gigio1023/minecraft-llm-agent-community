\subsection{Performance Variance and Pass\texorpdfstring{@}kk Rate}
\label{appx:analyses:pass_at_k}
Since running SWE-agent on SWE-bench can be rather expensive, we perform, all results, unless otherwise stated, are reported using a pass@\num{1} metric (\% Resolved).
However, we also test our main SWE-agent configuration for a higher number of runs to test the variance and pass$@k$ performance for $k\in\{3, 6\}$.
These results are shown in Table~\ref{tab:passk}, suggesting that average performance variance is relatively low, though per-instance resolution can change considerably.
