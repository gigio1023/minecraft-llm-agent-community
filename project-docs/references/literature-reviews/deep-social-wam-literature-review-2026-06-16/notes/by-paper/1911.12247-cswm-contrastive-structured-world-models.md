# C-SWM: Contrastive Learning of Structured World Models

- title: Contrastive Learning of Structured World Models (C-SWM)
- authors: Thomas Kipf, Elise van der Pol, Max Welling
- year: 2020
- venue: ICLR 2020
- arxiv_id: 1911.12247
- urls: paper https://arxiv.org/abs/1911.12247 , code https://github.com/tkipf/c-swm
- source_availability: latex (papers/latex/1911.12247/main.tex)

## Primary-source facts (what the paper states)

- Research question: can a structured, object-factored world model be learned from raw pixels
  WITHOUT a pixel-reconstruction loss, so that it predicts state transitions and generalizes to
  unseen object configurations?
- Model architecture (4 parts, Figure 1 in the paper):
  1. CNN object extractor `E_ext`: outputs K feature maps, each read as one object mask / slot.
  2. MLP object encoder `E_enc` (shared weights across slots): maps each mask to an abstract
     per-object state `z_t^k` in R^D.
  3. GNN relational transition model `T`: predicts per-object deltas via one round of message
     passing on a fully-connected scene graph. Edge update `e_t^{(i,j)} = f_edge([z_t^i, z_t^j])`,
     node update `dz_t^j = f_node([z_t^j, a_t^j, sum_{i!=j} e_t^{(i,j)}])`, then
     `z_{t+1} = z_t + dz_t`. Cost is O(K^2) in slots.
  4. Object-factored contrastive loss.
- Input / output: input is 50x50 RGB images (two stacked frames for Atari and physics);
  internal state is a SET of object latent vectors, NOT pixels and NOT a single global latent.
  Output is the predicted next-state object set in latent space (no decoder back to pixels).
- Action space: per-object actions, represented as one-hot vectors (zeros if no action applies
  to that object). The factored action space `A = A_1 x ... x A_K` lets each object be acted on
  independently.
- Training objective: a TransE-style (translational graph embedding) energy-based hinge loss.
  Energy of a positive triple `H = (1/K) sum_k d(z_t^k + T^k(z,a), z_{t+1}^k)`; negative energy
  `H~` uses a random state from the experience buffer; loss `L = H + max(0, gamma - H~)`,
  gamma = 1. The action is modeled as a learned translation conditioned on source state. NO
  pixel reconstruction term. Trained off-policy on a buffer of `(s_t, a_t, s_{t+1})` triples
  collected by a random policy.
- Evaluation protocol + metrics: ranking metrics in LATENT space, Hits at Rank 1 (H@1) and
  Mean Reciprocal Rank (MRR), at 1, 5, 10 prediction steps. Predicted next-state embedding is
  ranked against encoded true next state plus reference states from the buffer. Deliberately
  avoids pixel-reconstruction and downstream-task proxies.
- Datasets: two grid worlds (2D shapes, 3D blocks) with independently movable blocks, two Atari
  games (Pong, Space Invaders), and a 3-body gravitational physics sim. 1000 episodes x 100
  steps (grid), 1000 x 10 (Atari), 5000 x 10 (physics) for training.
- Key results: near-perfect (H@1 ~ 99-100% at 10 steps) on the structured grid worlds; large
  margins over autoencoder/VAE world-model baselines whose accuracy collapses at 10 steps
  (e.g. 2D shapes World Model (AE) 10-step H@1 = 6.5% vs C-SWM 99.9%). Ablations show both the
  object factorization and the contrastive (vs pixel) loss are necessary; removing the GNN
  (object-wise MLP) destroys interaction prediction. Atari is much harder and high-variance.
- Stated limitations:
  - Instance disambiguation: the feed-forward CNN extractor cannot separate multiple identical
    objects; needs iterative slot binding (the gap Slot Attention later fills).
  - Determinism + Markov: no stochastic transitions; single-step triples, no memory/RNN, so no
    hidden-state or partial observability.

## Interpretation (my reading, labeled inference)

- C-SWM is the canonical demonstration that a world model can predict in OBJECT + RELATION
  space and be scored WITHOUT pixels. The transition is `p(z' | z, a)` over a structured latent
  set, which is exactly the "implicit physical representation" branch of the WAM survey's
  definition (`<ROOT>/papers/latex/2605.12090/`): forecast future state implicitly, here as
  structured latents.
- The TransE analogy is the load-bearing idea for this project: a transition is a typed edge
  `(source_state, action, target_state)`, scored by an energy. A Minecraft social transition
  `lend_item(alice, bob, pickaxe)` is literally an entity-relation-entity triple. C-SWM shows
  this framing trains and generalizes on structured domains.
- The ranking metric (H@1/MRR over candidate next states) is a ready-made template for scoring a
  social-material transition predictor against logged ground-truth deltas, without needing a
  generative renderer.

## Mechanically useful vs research contribution

- Mechanically useful for this repo:
  - The object-factored state + GNN-delta + energy-ranking pattern as the shape of a structured
    predictor: encode typed Minecraft+social state as a set of entity vectors, predict per-entity
    deltas with message passing, score by ranking the true post-state above corrupted ones.
  - The contrastive (no-decoder) objective: avoids the cost and failure modes of pixel
    reconstruction, which is irrelevant to a Mineflayer runtime that already owns symbolic truth.
- What this repo should AVOID / would be overclaim:
  - C-SWM discovers objects from PIXELS unsupervised; the project does not need that (state is
    already typed by the runtime). Borrow the transition/scoring idea, not the perception stack.
  - C-SWM is deterministic + Markov + non-social. Claiming it "does social state" would be false;
    no possession, obligation, trust, or memory is modeled. It is a representation template only.

## WAM layer(s) informed

- Cross-cutting representation footing. Most directly Physical (object positions/interactions)
  and Material (per-object factored state = per-item/possession state). The triple-scoring idea
  extends conceptually to Social/Institutional, but the paper itself only demonstrates physical.
