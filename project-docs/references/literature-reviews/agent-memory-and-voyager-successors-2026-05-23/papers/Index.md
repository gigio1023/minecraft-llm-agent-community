# Literature Research: AI Agents, Memory, and Cognitive Science

This directory contains deep-dive literature reviews exploring the intersection of Minecraft as an AI environment, modular memory architectures, and cognitive science paradigms.

## 📚 Core Research Areas

### [Minecraft AI Agents](Minecraft-Agents.md)
*Focuses on papers treating Minecraft as an embodied AI testing ground.*
Evaluates historical baselines (like Voyager and Ghost in the Minecraft) and contrasts their design limitations—which were heavily constrained by older, expensive, short-context LLMs—against modern reasoning models.

### [AI Memory Systems](AI-Memory-Systems.md)
*Focuses on how agents persist, retrieve, and synthesize information.*
Explores factual vs. experiential vs. working memory, vector DB constraints, and new benchmarks like AMA-Bench. Investigates how persistence semantics must differ across different knowledge layers.

### [Cognitive Science in AI](Cognitive-Science-AI.md)
*Focuses on applying human cognitive templates to LLM orchestration.*
Examines Dialogue-as-Simulation, Drama Managers, Tulving's trichotomy of memory, and architectures like CoALA and JEPA. Evaluates how these templates can be engineered into reliable code boundaries.
