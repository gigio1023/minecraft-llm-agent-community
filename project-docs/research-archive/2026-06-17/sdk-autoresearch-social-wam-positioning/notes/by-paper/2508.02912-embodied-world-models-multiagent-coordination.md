## Primary-source facts (abstract-verified)

- MARL question: engineer a communication protocol vs learn it end-to-end, under partial
  observability.
- Two strategies compared on a cooperative task-allocation problem:
  - Learned Direct Communication (LDC): protocol learned end-to-end.
  - Intention Communication: an engineered inductive bias via a compact learned world model,
    the Imagined Trajectory Generation Module (ITGM), which uses the agent's own policy to
    SIMULATE FUTURE STATES; a Message Generation Network compresses the plan into a message.
- Substrate: a grid world ("a canonical abstraction for embodied AI"), scaling complexity.
- Finding: the engineered, world-model-based approach beats emergent communication on
  performance, sample efficiency, scalability as complexity grows.
