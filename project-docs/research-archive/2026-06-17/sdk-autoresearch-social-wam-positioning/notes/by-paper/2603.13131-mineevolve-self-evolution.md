## Primary-source facts (abstract-verified)

- Pipeline of four named roles:
  - Monitor: converts each subgoal execution into TYPED feedback - state changes, inventory
    changes, failure types, progress signals, stagnation indicators.
  - Inducer: derives reusable skills from successful executions and remedies from
    failed/stagnant ones.
  - Curator: validates, merges, filters, retrieves knowledge entries.
  - Adaptor: repairs the unfinished plan under repeated failure or stagnation.
- Substrate: Minecraft, long-horizon embodied tasks. Code stated as available.

## Cross-product placement (the Lane B point)

Axes hit: Minecraft (yes), self-improvement loop (yes, self-evolution with accumulated
knowledge). Not a predictive world model in the WAM sense (no co-generated next-state `o'`
driving the loop; Monitor reads actual state, it does not predict it). No social axis.

Why this is the most INSTRUCTIVE Minecraft+self-improvement comparison for the repo:
- Its success signal is TYPED in-world feedback (state/inventory changes, failure types,
  stagnation) rather than an LLM self-judge over prose. That is much closer to the repo's
  verifier discipline than Voyager's GPT-4 self-critic or EvolvingAgent's self-verification.
  So MineEvolve is evidence that a Minecraft self-improvement loop CAN be grounded on typed
  state deltas - which is exactly the mechanism the repo wants, applied to physical/task
  competence.

What it MISSES relative to the 4-way frame:
- SOCIAL: none. Single agent, task-completion framed; no possession/claim/obligation/trust,
  no second actor, no post-goal social continuation.
- WORLD MODEL (predictive): no advisory `p(o'|o,l)` predictor; the loop reacts to observed
  deltas rather than predicting-then-checking them. So it does not occupy the
  "predict-and-verify" cell either.

Net: MineEvolve occupies Minecraft + self-improvement with a typed-feedback (verifier-like)
signal, single-agent and physical. It is the closest precedent for the repo's "loop grounded
on typed in-world deltas, not a self-judge", but it carries neither the social-material layer
nor an advisory predictive WAM.
