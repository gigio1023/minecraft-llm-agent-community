# Machine Theory of Mind (ToMnet)

- title: Machine Theory of Mind
- authors: Neil C. Rabinowitz, Frank Perbet, H. Francis Song, Chiyuan Zhang, S. M. Ali Eslami, Matthew Botvinick (DeepMind / Google Brain)
- year: 2018
- venue/source: ICML 2018; arXiv 1802.07740
- arxiv_id: 1802.07740
- urls: paper https://arxiv.org/abs/1802.07740 ; hf https://huggingface.co/papers/1802.07740 ; no official code release found
- source availability: LaTeX (`papers/latex/1802.07740/longform.tex`), read in full.

## Primary-source facts (what the paper states)

- Research question: can a machine learn, *autonomously and from behavior alone*, to
  model other agents (their characteristics, future actions, and mental states),
  rather than being handed a hand-crafted generative model of agents to invert?
- Key framing move: theory of mind (ToM) here means building high-level predictive
  *models* of other agents (desires, beliefs, intentions) judged by usefulness for
  prediction and planning, not by 1-to-1 correspondence to the agent's true internal
  mechanism (Dennett's intentional stance). This deliberately departs from prior
  hand-built approaches (inverse RL, Bayesian ToM, game theory) by *learning* both
  the agent model and the inference procedure via meta-learning.
- Method: the ToMnet is a meta-learning architecture with three modules:
  - character net: parses a set of an agent's past episode trajectories into a
    character embedding e_char (a prior over that agent's behavior).
  - mental state net: parses the agent's current-episode trajectory so far into a
    mental-state embedding e_mental (a posterior, "what is it thinking now").
  - prediction net: conditions on both embeddings + current state to predict
    next-step action probabilities, which objects it will consume, and successor
    representations (discounted future state occupancy).
- Setup: a family of single-agent gridworld POMDPs; reward/discount/observation
  functions are attached to *agents*, not the world, so a population of varied
  agents (random, algorithmic, deep-RL) can be placed in the same maze. The observer
  sees full POMDP state + the agent's actions, but NOT the agent's parameters,
  reward function, policy, or identity. N_past (number of past episodes seen) can
  vary, even be zero.
- Results, in order of increasing ToM richness:
  1. For random agents the ToMnet learns to approximate Bayes-optimal hierarchical
     inference over agent characteristics.
  2. For algorithmic agents it infers goals (effectively few-shot inverse RL) and
     how they trade off costs vs rewards.
  3. For deep-RL agents it characterizes "species" (e.g. different fields of view),
     forms abstract embeddings, and can discover new behavioral abstractions.
  4. False belief (implicit): the world has low-probability "swap events" (objects
     permute) that the agent only sees if within its field of view. Trained only to
     predict overt behavior, the ToMnet learns to predict that an agent whose
     preferred object moved out of sight will persist on a policy founded on a FALSE
     belief about where the object is. It passes a gridworld "Sally-Anne" test.
     Strikingly, the ToMnet learned these false-belief statistics even when it had
     never observed any swap events during its own training (swap p=0 training).
  5. False belief (explicit): a supervised variant of the ToMnet can directly
     predict the agent's belief state b = p(object location | history), revealing
     false beliefs explicitly, and infers what each agent can see (its observability)
     from behavior alone, so it predicts which agents are more prone to false beliefs.
- Stated limitations (theirs): experiments are simple gridworlds; the observer has
  full observability (they did not limit what the observer itself can see); they did
  not yet situate the ToMnet inside an acting agent in a multi-agent task (it is a
  passive observer). They flag scaling to rich 3D, limited-observer inference, richer
  agent dimensions (animate/inanimate, prosocial/adversarial, reactive/planning), and
  embedding the ToMnet in a decision-making agent as future work.

## Interpretation (my reading, labeled inference)

- ToMnet is the foundational "computational theory of mind" result: it shows that
  predicting another agent's *behavior* and predicting that agent's (latent, possibly
  false) *belief* are learnable from observed trajectories, and that the false-belief
  signature can fall out of pure behavior prediction. For this project, this is the
  conceptual proof that a Social WAM target ("predict what Bob will do, and what Bob
  believes about the world") is a coherent, learnable prediction target, not mysticism.
- The character-embedding (prior over an agent) vs mental-state-embedding (posterior
  this episode) split maps directly onto a repo distinction the project already cares
  about: a persistent per-actor model (relationship history, reputation) vs an
  in-the-moment read of the current situation.
- Important caveat for honesty: ToMnet's belief read is *amortized neural inference
  trained over a population*, not an LLM zero-shot claim. It is evidence that the
  capability is learnable, NOT evidence that an LLM has it. The LLM-ToM debate
  (Kosinski 2302.02083 vs Ullman 2302.08399) is a separate, contested question.

## Mechanically useful vs research contribution

- Mechanically useful for this repo:
  - The prediction target schema: for each other actor, predict (a) next action /
    action distribution, (b) which resources/affordances they will consume or claim,
    (c) a "successor-representation"-like forecast of where they will go / what they
    will hold. These are exactly typed social-material deltas a Social WAM would emit.
  - The observability-grounded false-belief test: in Minecraft, "Bob did not see Alice
    move the chest, so Bob still believes the iron is in the old chest" is a directly
    loggable, verifiable scenario (field of view -> belief -> predicted action). This
    is a concrete, world-anchored ToM evaluation the repo could build.
  - The prior/posterior decomposition (persistent character model + per-episode read).
- What this repo should avoid / overclaim risk:
  - Do NOT replicate ToMnet's *method* (meta-trained neural net over millions of agent
    episodes); the repo's actors are LLM-proposed turns, not gradient-trained policies.
    Borrow the prediction target and the false-belief evaluation design, not the training.
  - Do NOT claim the runtime "has a theory of mind." The defensible claim is "the WAM
    predicts another actor's next action / belief-consistent behavior, and that
    prediction is scored against verifier evidence." Internal mental state is not observable.

## WAM layer(s) informed

- **Social WAM** primarily (predicting another agent's future action and belief state).
- **Institutional WAM** secondarily (a population-level prior over agent "species" /
  roles is the seed of role and convention modeling).
- Depends on **Physical WAM** truth: a belief prediction ("Bob thinks the iron is
  here") is only meaningful once physical facts (object locations, field of view) are
  reliable, exactly the project's "physical-before-social" dependency.

## Reproducibility

- Partial: the architecture and gridworld design are described in detail and are
  reproducible in principle, but no official code/weights release was found, and the
  agent populations are bespoke. Transfers to this repo as *design and evaluation
  concept*, not as runnable code.
