%\llm{} coding abilities are almost always evaluated using the \humaneval{} benchmark, overindexing the evaluations to simple code generation problems possibly contaminated in current \llms{}.
In this work, we propose \livecodebench{}, a new benchmark for evaluating \llms{} for code.
Our benchmark mitigates contamination issues in existing benchmarks by introducing live evaluations and emphasizing scenarios beyond code generation to account for the broader coding abilities of \llms{}.
\livecodebench{} is an extensible framework, that will keep on updating with new problems, scenarios, and models.
Our evaluations reveal novel findings such as contamination detection and potential overfitting on \humaneval{}.
We hope \livecodebench{} with serve to advance understanding of current code \llms{} and also guide future research in this area through our findings.
