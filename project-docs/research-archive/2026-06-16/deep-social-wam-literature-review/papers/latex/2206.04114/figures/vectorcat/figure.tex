\caption{
Director uses a variational autoencoder to turn its state representations into discrete tokens that are easy to select between for the manager policy. 
For this, we use the vector of categoricals approach of DreamerV2 \citep{hafner2020dreamerv2}. 
For a given input vector, the encoder outputs a matrix of logits of $L$ latent dimensions with $C$ classes each. 
We sample from the logits and one-hot encode the result to obtain a sparse matrix of the same shape as the logits. 
To backpropagate gradients through this step, we simply use the gradient with respect to the one-hot matrix as the gradient with respect to the categorical probabilities \citep{bengio2013straight}. 
The matrix is then flattened, resulting in a sparse representation with $L$ out of the $L \times C$ feature dimensions set to one.
}
\label{fig:vectorcat}
